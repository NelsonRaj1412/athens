Ensure you're using the `useResponsiveSidebar` hook and applying the correct CSS classes.\n\n### Issue: Forms too cramped on mobile\n**Solution:** Use `ResponsiveFormWrapper` with `complexity=\"complex\"` for complex forms.\n\n### Issue: Tables not scrolling properly\n**Solution:** Wrap tables with `ResponsiveTableWrapper` component.\n\n### Issue: Buttons not stacking on mobile\n**Solution:** Use `ResponsiveFormActions` or add `responsive-button-group` class.\n\n### Issue: Text too small on mobile\n**Solution:** Use responsive typography classes like `responsive-title-lg`.\n\n## Performance Considerations\n\n1. **CSS-only animations** on mobile for better performance\n2. **Debounced resize handlers** to prevent excessive re-renders\n3. **Optimized
